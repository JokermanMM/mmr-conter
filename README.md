# Dota 2 MMR Counter Bot

Этот бот отслеживает ваши игры в Dota 2 и присылает уведомления о изменении ММР.

## Инструкция по запуску:

1.  Установите зависимости:
    ```bash
    pip install -r requirements.txt
    ```
2.  Запустите бота:
    ```bash
    python main.py
    ```

## Как пользоваться:

1.  В Telegram напишите боту команду `/start`.
2.  Укажите свой Steam ID командой `/set_id <ваш_id>`.
    *   *Пример: `/set_id 12345678`*
    *   ID можно взять из ссылки на ваш профиль в Dotabuff.
3.  **Важно**: В настройках Dota 2 должна быть включена опция **"Expose Public Match Data"** (Общедоступная история матчей).

## Особенности:
- Бот использует **Stratz GraphQL API** для получения точного ММР (`afterMmr`).
- Уведомления приходят автоматически после каждой игры (проверка каждые 3 минуты).
- Красивое оформление с результатами матча и статистикой KDA.
